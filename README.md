# SmartDelivery
SmartDelivery — Интерфейс сервиса доставки. Что сделано: — UI для отображения услуг доставки — адаптивная верстка — логичная структура страниц. Калькулятор стоимости доставки Интерактивный калькулятор с расчетом по расстоянию, весу, типу груза и дополнительным услугам. Мгновенный пересчет, детализация цены, адаптивный дизайн.
